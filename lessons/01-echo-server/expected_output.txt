# Lesson 01: Echo Server - 期待される出力

## サーバー側（ターミナル1）

```
$ ./server
サーバーを起動しました。ポート 8080 で接続を待っています...
クライアントが接続しました: 127.0.0.1
受信: Hello, TCP/IP!
受信: This is a test message.
クライアントが切断しました
次の接続を待っています...
```

## クライアント側（ターミナル2）

```
$ ./client
サーバーに接続しました。
メッセージを入力してください（終了するには 'quit' と入力）:
> Hello, TCP/IP!
サーバーからの返答: Hello, TCP/IP!
> This is a test message.
サーバーからの返答: This is a test message.
> quit
接続を終了します。
```

## エラー時の出力例

### サーバーが起動していない場合

```
$ ./client
エラー: サーバーへの接続に失敗しました
原因: サーバーが起動していないか、ポート番号が間違っています
対処法:
  1. サーバー（./server）が起動しているか確認してください
  2. ポート番号 8080 が正しいか確認してください
  3. ファイアウォールがブロックしていないか確認してください
```

### ポートが既に使用中の場合

```
$ ./server
エラー: ポートへのバインドに失敗しました
原因: ポート 8080 は既に使用中の可能性があります
対処法: 'lsof -i :8080' で使用中のプロセスを確認し、
        終了するか、別のポート番号を使用してください
```
